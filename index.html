⚙️ Configuración de Notificaciones

Completá las credenciales de EmailJS para habilitar el envío de recordatorios.
También podés configurar el webhook de Make.com para envíos automáticos el 1° de cada mes.

📧 EmailJS ⚠ Sin configurar

En EmailJS → Email Services → copiá el ID del servicio conectado a Gmail.
En EmailJS → Email Templates → copiá el ID de la plantilla de recordatorio.
En EmailJS → Account → Public Key.
Variables a usar en tu plantilla EmailJS:
{{to_name}} — Nombre del auditor  |  {{to_email}} — Email destino  |  {{proceso}} — Área(s) a auditar  |  {{mes}} — Mes en curso

🔄 Make.com (automático) ⚠ Sin configurar

Creá un escenario en Make.com con módulo "Webhooks → Custom webhook", copiá la URL aquí.
Cada vez que guardás el calendario, los datos se sincronizan con Make.com automáticamente.
